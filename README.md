# Mintlify Landing Page Clone

This project is a **desktop-first visual recreation of the Mintlify documentation landing page**, built strictly using **HTML and CSS only**.  
The goal of this project is to replicate the **layout, typography, spacing, and visual hierarchy** of the original Mintlify website as closely as possible.

> ⚠️ This project focuses on **UI structure and visual accuracy**, not interactivity or backend functionality.

## ✅ Sections Recreated

The following sections were recreated based on the original Mintlify website:

1. **Top Navigation Bar**
   - Logo
   - Documentation links
   - Primary call-to-action button
   - Clean light/dark background styling

2. **Hero Section**
   - Main heading and tagline
   - Supporting description
   - Primary CTA buttons
   - Product/illustration preview

3. **Features Section**
   - Grid-based layout
   - Icon + heading + description blocks
   - Consistent spacing and alignment

4. **Developer Experience Section**
   - Two-column layout (text + image)
   - Emphasis on documentation simplicity

5. **Integration / Tooling Highlights**
   - Card-based layout
   - Short descriptions with visuals

6. **Use Cases / Benefits**
   - Simple text and image-based cards

7. **Call-To-Action Section**
   - Prominent heading
   - Primary action button

8. **Footer**
   - Multi-column link layout
   - Documentation, company, and resource links
